# X運用トレンドリサーチ（2026年3月）

## 1. アルゴリズムの大変革 — Grok AI統合

2026年、XのアルゴリズムはGrok AIの導入で根本的に変わった。

- Grokが**毎日1億件以上の投稿**を読み込み、ユーザーの興味・トレンドに基づき最適コンテンツをマッチング
- 2026年1月20日、アルゴリズムの一部が**GitHub上でオープンソース化**
- 「なんとなく投稿する」→「**アルゴリズムを理解して設計する**」時代へパラダイムシフト

## 2. エンゲージメントの評価ポイント制

| アクション | スコア倍率 | 備考 |
|-----------|-----------|------|
| リツイート | **20x** | 圧倒的に高い。RT誘発が最重要 |
| 引用ポスト | **15x** | 新しい会話を生む点が評価される |
| 画像・動画付き | **2x** | 必ずメディアを添付すべき |
| リプ返し | **大幅UP** | リプが来たら即返信でスコア加速 |

## 3. 投稿の寿命 — 時間減衰（Time Decay）

- 投稿は**6時間ごとに可視性スコアが半減**する
- 最初の2〜3時間のエンゲージメントが投稿の生死を決める
- → **投稿直後のリプ対応が超重要**

## 4. フォロワー数 < エンゲージメント品質

- フォロワー総数はリーチの予測因子として弱くなった
- **最初の数百viewでの反応品質**が投稿の拡散を決定する
- 小さくても濃いコミュニティが有利

## 5. 2026年の勝ちパターン

### コンテンツ形式
| 形式 | 効果 | 備考 |
|------|------|------|
| スレッド投稿 | ◎ | クリック読み進めでスコア大幅UP |
| 60秒以内の動画 | ◎ | 動画は前年比29%成長、最大の成長レバー |
| 短いテイク（一言系） | ○ | エンゲージメント誘発に最適 |
| 長文投稿 | ○ | 深掘りコンテンツとして機能 |

### 運用のコツ
- **ハッシュタグは1〜2個まで**（3個以上はエンゲージメント低下）
- **テーマの一貫性**を持たせ、定期投稿を継続
- **エンゲージメント率は1.32→1.58に上昇**（RT+35%、リプ+21%）
- リプ欄が「本当のマーケティング」の場になっている

### 個人開発者・SaaS向け
- 代表個人アカウント × 企業アカウントの**連携運用**が効果的
- 週1の**スレッド形式「業界レポート」**が高RT（平均100以上の事例あり）
- リプ丁寧返信 → DM経由の商談化が王道パターン
- ある SaaS企業は X運用6ヶ月で**月間リード獲得数3倍**

## 6. 既存Tipsとの統合分析

保存済みTips（`tips/x-posting-strategy.md`）との照合:

| Tips原則 | アルゴリズム裏付け |
|---------|-------------------|
| ①市場を広くとれ | ✅ 最初の数百viewの反応品質が拡散を決める → 母数が大きいテーマほど初動が良い |
| ②言い切れ | ✅ RT(20x)・引用(15x)が最高スコア → 賛否が分かれる言い切りがRT・引用を誘発 |
| ③人でなく考え方をぶつけろ | ✅ リプ欄が本当のマーケティングの場 → 建設的な議論がリプ数を増やしスコアUP |
| フォロワーリサーチ | ✅ フォロワー数<エンゲージメント品質の時代 → 少数でも刺さる投稿が勝つ |
